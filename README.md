# form-factory
> 表单页面可视化、拖拽编辑，快速生成表单页面

## Build Setup
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9999
npm run dev

# build for production with minification
npm run build

# build report
npm run build:report
```

* 数字输入
